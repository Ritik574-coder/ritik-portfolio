# Data Engineering Portfolio App

This app has been transformed from a generic visual template into a recruiter-focused Data Engineering portfolio for Ritik Kumar.

## Content Rules

The site should position Ritik as:

- Data Engineer
- Data Platform Builder
- Data Warehouse Engineer
- Analytics Engineering Practitioner

Avoid positioning the site around frontend development, web development, entertainment, events, or generic software engineering.

## Source of Truth

Use `src/data/portfolio.ts` for portfolio content. It contains typed data for:

- Profile
- About
- GitHub proof
- Projects
- Skills
- Certifications
- Resume asset path

## Asset Rules

Use assets from `public/assets`.

- Profile image: `public/assets/profile/ritik-kumar-portrait.png`
- Resume PDF: `public/assets/resume/ritik-resume.pdf`
- Certificate PDFs: `public/assets/certificates`

## Design Direction

The visual language should remain premium, dark, recruiter-friendly, and data-platform oriented:

- Pipeline lines
- Warehouse/data model language
- Subtle glass panels
- Calm cyan, teal, blue, and amber accents
- Minimal professional motion

Avoid crypto, gaming, entertainment, album, event, and generic landing-page aesthetics.
